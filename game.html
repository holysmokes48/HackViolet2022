Computational thinking involves taking a problem and breaking it down into simple steps, allowing us to design potential solutions. This allows us implement these solutions in ways that a computer or human can understand.

x

To give you a real life example of this we created a game where you must navigate your character through a maze using four commands.

x

Character.up()

x

Character.down()

x

Character.left()

x

Character.right()

x

If you hit the sides of the maze by using the wrong command the game will be over.
